BISEC Dashboard Web Application is designed based on bootstrap 5 & built on React along with Vite.

## Quick Start
BISEC Dashboard Web Application requires JS, CSS 3, HTML 5, Bootstrap 5, jQuery, React, Vite knowledge for Implementation

### Method 1: Direct Download
[Dowload from Github](https://github.com/ptsprionjit/bisec/tree/main)

### Method 2: Using NPM
Start working with the design system
1. Install Dependency
```
npm install
```

2. Run on development 
```
npm run dev
```
3. Build for production
```
npm run build
```
